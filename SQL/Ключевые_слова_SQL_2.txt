LIKE~используется в предложении WHERE для поиска указанного шаблона в столбце.Вы можете использовать два подстановочных знака с ????: % - Представляет ноль, один или несколько символов. _ - Представляет собой один символ (MS Access использует знак вопроса (?) вместо)
AVG()~Среднее значение по столбцу
COUNT()~число строк в столбце
MAX()~Наибольшее значение в столбце
MIN()~Наименьшее значение в столбце
SUM()~Сумма значений столбца
DISTINCT~При вычислении учитываются только уникальные значения
GROUP BY~можно использовать в операторе SELECT для сбора данных по нескольким записям и группировки результатов по одному или нескольким столбцам.
ALL~ALL используются с предложением WHERE или HAVING.Оператор ??? возвращает true, если все значения подзапроса удовлетворяют условию.
HAVING~позволяет фильтровать результат группировки, сделанной с помощью команды group by.
